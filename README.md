# ContactVault
**Portfolio piece • Full‑stack • Node.js • React • MongoDB • Redis • RabbitMQ • Docker • Tilt**

ContactVault is a deliberately over‑engineered contact manager — a full‑stack application that manages people (contacts) with their phone numbers, emails and addresses. While the domain is simple, the architecture is intentionally heavy: it mimics a production micro‑services setup with caching, asynchronous logging, container orchestration, and comprehensive documentation/testing.

It was built as a portfolio project to demonstrate real‑world skills that go far beyond basic CRUD — showing that I can work with modern backend infrastructure, thoughtful developer tooling, and a polished frontend, all wired together in a single, runnable repository.
